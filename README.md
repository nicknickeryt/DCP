# DCP - DragonCraftPomoc
Plugin wyświetlający pomoc dotyczącą serwera Minecraft, napisany dla serwera DragonCraft. Testowany na wersji 1.12.2.

## Komendy
/pomoc - wyświetla pomoc<br>
/rlpomoc - przeładowuje konfigurację
